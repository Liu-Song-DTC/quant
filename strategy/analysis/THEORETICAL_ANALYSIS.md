## 2. 核心问题分析

### 2.1 IC vs 胜率的矛盾

**发现**：因子单调性良好（IC=0.89），但买入胜率只有49.64%

**原因**：A股收益分布右偏
- 均值 = 1.04%
- 中位数 = 0.00%
- 正收益比例 = 48.5%

**结论**：策略选的是"高期望收益"股票，但不保证"高上涨概率"

### 2.2 理论Sharpe上限

根据基本方程：
```
Sharpe = IC × √N × (σ_factor / σ_portfolio)
```

当前参数：
- IC = 2.7%
- N = 8（持仓数）
- σ_factor/σ_portfolio ≈ 2

理论Sharpe上限 ≈ 0.15

但实际Sharpe = 0.484，说明策略捕获了理论模型外的alpha。

## 3. 优化路径（理论推导）

### 路径1：提高IC（优先级：高）

**目标**：IC从2.7%提升到5%+

**方法**：
1. **行业中性化**
   - 当前问题：行业因子混杂在alpha因子中
   - 解决：每个行业内单独做因子排名，消除行业bias

2. **因子正交化**
   - 当前问题：多个因子高度相关，信息冗余
   - 解决：使用正交化提取独立alpha

3. **IC加权**
   - 当前问题：等权平均，高质量因子被稀释
   - 解决：用IC的平方根加权

### 路径2：提高胜率（优先级：中）

**目标**：胜率从49.64%提升到52%+

**方法**：
1. **更严格的买入阈值**
   - 当前：factor_value > 0.25
   - 建议：factor_value > 0.5（提高门槛，减少弱信号）

2. **多因子交叉验证**
   - 只在多个因子同时看多时买入
   - 减少假阳性

3. **市场状态过滤**
   - 熊市降低仓位或空仓
   - 避免系统性风险

### 路径3：风险控制（优先级：中）

**目标**：降低回撤，提高夏普

**方法**：
1. **降低集中度**
   - 当前：最大权重53.74%
   - 建议：单只股票权重上限15%

2. **增加持仓数**
   - 当前：8只
   - 建议：10-15只

## 4. 不推荐的优化（可能过拟合）

- ❌ 继续放宽因子阈值（已接近极限）
- ❌ 针对特定行业/时间优化
- ❌ 增加过多参数
