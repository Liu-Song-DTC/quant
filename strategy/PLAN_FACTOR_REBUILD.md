# 因子库重建计划

## 背景

当前因子体系核心问题：
- 熊市IC = -0.023（因子是反指），导致2022年买入准确率45.9%（不如扔硬币）
- 非牛市年份买入准确率46-53%，低于55%目标
- 收益依赖敞口控制（少买少亏）而非信号质量（高胜率赚钱）

## 防过拟合约束

- **因子选择窗口**：2016-2020（与ML训练窗口一致），2021-2026完全不参与因子选择
- **回测验证窗口**：2021-2026，纯样本外，因子选择阶段不可见
- **统计显著性**：Bonferroni校正（α = 0.05 / 候选因子总数）
- **幸存者偏差**：实盘与回测使用同一候选池，偏差一致，不是问题

## 目标

1. 全市场周期 IC > 0.05（当前 ~0.02）
2. 熊市 IC > 0.02（当前 -0.023）
3. 买入准确率 > 55%（当前牛市56%、熊市46%）

## Phase 1: 审计现有因子（1-2天）

### 1.1 按市场状态分解 IC
- 计算每个因子在 bull/neutral/bear/bear_risk/bear_risk_fast 五种状态下的 IC
- 只使用2016-2020数据计算
- 按行业分组，找出哪些行业在熊市中因子完全失效
- 输出：因子×市场状态 IC 矩阵

### 1.2 因子归因分析
- 现有127个因子分为：动量类、波动率类、基本面类、量价类、技术类
- 每类在熊市中的表现差异
- 找出"拖后腿"的因子类别（预期：动量类在熊市最差）

### 1.3 相关性矩阵
- 计算因子间相关性，剔除冗余因子（|ρ| > 0.7）
- 保留互不相关的独立信号源

## Phase 2: 候选因子池（2-3周）

### 2.1 A股特定因子
- **资金流向类**：北向资金/融资融券的行业级信号（已部分实现）
- **股东行为类**：增减持、回购、股权激励
- **机构行为类**：基金重仓变化、龙虎榜机构席位
- **情绪类**：涨停板/跌停板扩散度、市场宽度

### 2.2 熊市防御因子（重点）
- **质量类**：ROE稳定性、现金流/利润比、股息率
- **低风险类**：低beta、低波动率稳定性（已有部分）
- **反转类**：短期超卖反弹、波动率均值回归
- **流动性类**：Amihud非流动性、换手率稳定性（已有）

### 2.3 跨资产信号
- 国债收益率变化 vs 股票行业轮动
- 商品价格（铜/油/黄金）vs 相关行业
- 汇率变化 vs 出口/进口依赖行业

### 2.4 候选因子筛选标准
- 只使用2016-2020数据
- 每个候选因子必须通过：
  1. IC > 0.02 至少在一个市场状态下
  2. 样本量 > 5000（统计显著）
  3. 无前视偏差（滞后至少1天）
  4. 覆盖至少60%股票（避免样本选择偏差）
  5. Bonferroni校正通过（α = 0.05 / N，N为候选因子总数）

## Phase 3: 因子验证（1-2周）

### 3.1 时间序列交叉验证（只用2016-2020）
- 分两段交叉验证：2016-2017训练→2018验证，2018-2019训练→2020验证
- 两段均IC>0.02的因子才算稳健
- 淘汰只在单段有效的因子
- 2021-2026完全不被触碰，留作最终回测

### 3.2 跨行业稳定性
- 每个因子在至少5个行业IC>0.02
- 行业间离散度 < 2×均值（避免个别行业驱动）

### 3.3 实盘模拟（最终验证，2021-2026）
- 在现有回测框架中添加新因子
- 保持其他逻辑不变，只替换因子
- 此时才第一次看到2021-2026的表现
- 对比新老因子的年度表现（收益/MDD/准确率）

## Phase 4: 分市场因子配置（1-2周）

### 4.1 构建分市场因子集
- 基于Phase 1-3的验证结果
- 为每个行业配置三套因子：bull_factors、neutral_factors、bear_factors
- 每套3-5个因子，权重基于IC比例

### 4.2 因子权重方案
- 方案A：固定权重（IC比例线性加权）
- 方案B：动态权重（滚动24个月IC更新权重）
- 方案C：状态切换（根据市场状态切换因子集+固定权重）
- 回测对比三种方案的稳定性

### 4.3 与现有ML预测器集成
- ML预测器当前使用全因子训练
- 分市场状态的ML模型（bull/bear分开训练）
- 评估是否需要分状态ML（预期：熊市ML模型能捕捉非线性防御模式）

## Phase 5: 组合层适配（1周）

### 5.1 因子切换接口
- SignalEngine支持per-date因子切换（当前只支持per-industry静态配置）
- 根据市场状态（regime/bear_risk/trend_score）选择因子集

### 5.2 回测验证
- 逐步替换现有因子，观察每个阶段的增量
- 最终对比：新因子+敞口控制 vs 当前版本+敞口控制
- 评估指标：年度收益、MDD、Sharpe、买入准确率

## 时间线

| Phase | 内容 | 时间 | 依赖 |
|-------|------|------|------|
| P1 | 审计现有因子 | 1-2天 | 无 |
| P2 | 候选因子池 | 2-3周 | P1 |
| P3 | 因子验证 | 1-2周 | P2 |
| P4 | 分市场配置 | 1-2周 | P3 |
| P5 | 组合层适配 | 1周 | P4 |
| **总计** | | **5-9周** | |

## 优先级

1. **P1 + P2.2（熊市防御因子）** — 最高优先级，直接针对当前最大痛点
2. **P2.1（A股特定因子）** — 高优先级，差异化优势来源
3. P2.3-2.4 — 中等优先级，锦上添花
4. P3-P5 — 验证和集成，必须做但依赖前序完成

## 预期效果

保守估计：熊市IC从-0.023提升到+0.010，2022年买入准确率从45.9%提升到51%+，
配合现有敞口控制，2022年收益有望从-29%改善到-10%以内。
