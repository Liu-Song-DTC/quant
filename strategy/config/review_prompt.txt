## 核心要求

### 1. 专业性优先
- 分析要有理有据, 缠论理论+数据证据+图表配合
- 不只是说"背驰了", 要说清楚"为什么这里是背驰 -- K线价格创了新低, 但MACD柱子面积明显缩小, 黄白线也未同步新低, 符合第24课背驰标准"
- 每个判断都要有可验证的逻辑链条

### 2. 配图必须展示K线和MACD配合 (K线要尽可能真实)
K线图要模拟真实蜡烛图结构: yang线(涨)=阳实体 `|+|`, yin线(跌)=阴实体 `|#|`, 上下影线用 `|`。

个股结构图模板:

```
  {股票代码} {股票名称} | 收盘{价格} | {买点类型}

     K线(每根蜡烛含开/高/低/收)          MACD指标(柱面积对比)
     ^                                   ^
     | |    <- 上影线                    |  ##### [前段下跌MACD柱面积: 大]
     |#|    <- 阴线实体(收<开)           |  #####
     | |    <- 下影线                    |  ####
     |+|    <- 阳线实体(收>开)           |  ####
     |#|                                 |  ###
     |#|    <- 价格新低                  |  ##   [后段MACD柱面积: 明显缩小]
     |#|                                 |  ##
     |+|                                 |  #
     |+|                                 |  #
     +-*B1(底背驰)-------->              +-------------------->
       价格创新低, 但MACD柱面积明显缩小     MACD未同步新低 = 背驰成立(第24课)
```

K线绘图规则:
- 每根K线占3行: 上影线 | 实体 | 下影线
- 阳线(涨): 实体行为 `|+|`, 中间+代表实体中心(收盘高于开盘)
- 阴线(跌): 实体行为 `|#|`, 中间#代表实体被填充(收盘低于开盘)
- 十字星: `|+|` 或 `|o|`
- 连续K线要能看出趋势方向(上涨=阳线多, 下跌=阴线多)
- 中枢震荡区用 +---+ 框起来
- 买点/卖点用 *B1/*B2/*B3/*S1/*S2/*S3 标注

每种买/卖点的图解都要包含:
1. 左侧K线栏: 多根真实蜡烛图展示价格走势
2. 右侧MACD栏: #号表示MACD柱(红柱/绿柱), 柱高度表示面积大小
3. 底部标注: 说明价格行为与MACD行为的对比(为什么这里是背驰/突破/回踩)

### 附1: 第一类买点 (一买/B1) — 底背驰终结下跌趋势
- 第21课: 下跌趋势末端出现底背驰, 空头力竭
- 第24课: MACD柱面积明显缩小, 黄白线未同步新低
- K线+MACD图解(每根蜡烛=上影+实体+下影, yang=|+|, yin=|#|):
  ```
      K线(每根3行:影线/实体/影线)         MACD柱面积
      ^                                   ^
      | | <-上影                           |
      |#| <-阴体(收<开)                    |  #####  <-前段下跌, MACD柱面积大
      | | <-下影                           |  #####
      |+| <-阳体(收>开)                    |  #####
      |#|                                  |  ####
      | | <-价格新低                       |  ###
      |#| <-阴体                           |  ##    <-后段, MACD柱面积明显缩小

- 操作: MACD金叉确认后买入, 止损设在前低下方2-3%

### 附2: 第二类买点 (二买/B2) — 回踩不破一买低点
- 第21/49课: 一买后次级别回踩, 不创新低, 最安全的买点
- K线+MACD图解:
  ```
      K线                                MACD
      ^                                   ^
      | |                                 |
      |+|          | | <-回踩段阴线        |  ####         ### <-回踩MACD柱小
      |+|   | |    |#|                    |  ######    ####
      |+|   |+|    |#|                    |  ###########
      |+|   |+|    |+| <-止跌阳线         |  ###########
      +-*B1--------*B2----------->        +---------------------->
        一买底背驰   回踩不破B1低点         回踩时MACD回落但不破0轴
  ```
- 判定: B1后回踩, 最低点高于B1低点, 成交量萎缩, MACD未破0轴
- 操作: 回踩止跌+MACD再次金叉时加仓, 止损在B1低点下方

### 附3: 第三类买点 (三买/B3) — 突破中枢后回踩确认
- 第21/54课: 突破中枢上沿, 次级别回踩不跌回中枢
- K线+MACD图解:
  ```
      K线                                MACD
      ^                                   ^
      | |                        |+| <-阳 |               #### <-放量突破
      |+|  +------------+  |#|   |+|     |            ####
      |+|  |  中枢震荡   |  |#| <-回踩  |         ####
      |#|  | (多空平衡区) |  |+| <-不破  |      ####
      |+|  +------------+  |+|          |   ####
      |+|          |+|     |+|           |####

- 判定: 价格突破中枢上沿, 次级别回踩不跌回中枢区间, 成交量放大配合
- 操作: 放量突破中枢上沿+回踩确认后介入, 止损在中枢上沿下方
